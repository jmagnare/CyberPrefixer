MorbidHeadlines
=============

Twitter bot to tweet depressing/sad/morbid headlines. Idea adapted from [Molly](https://github.com/mollywhite)'s CyberPrefixer bot. I saw the morbid prefix she'd written to give the bot some tact, and I thought it deserved its own voice.

Twitter account @MorbidHeadlines
Profile picture from [ijustwanttobeperceivedthewayiam](http://www.flickr.com/photos/67776729@N06/) on flickr, via a [CC BY 2.0 license](http://creativecommons.org/licenses/by/2.0/deed.en).


## Requires ##
* [BeautifulSoup](http://www.crummy.com/software/BeautifulSoup/) for HTML parsing
* [tweepy](https://github.com/tweepy/tweepy) for Tweeting
